## 优化方向（保持功能不变）
1. **减少 WebView2 进程数**
   - 隐藏窗口时销毁 WebView，而不是仅 hide。
   - 合并 `main` 与 `taskbar`（若 UI 允许）。
2. **替换 PowerShell GPU 采集**
   - 使用 PDH / DXGI / PerfCounter 直接在 Rust 侧采集，避免常驻 PowerShell。
3. **有选择地回收 WebView2 Working Set**
   - 仅在窗口隐藏或空闲时回收，避免系统级全进程回收。
4. **前端内存优化**
   - 降低 DOM 复杂度、虚拟列表、及时销毁图表与大对象。

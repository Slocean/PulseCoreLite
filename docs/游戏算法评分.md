🎮 游戏评分算法逻辑文档

1. GPU 游戏力计算 (核心指标)
   不再硬写每个显卡的分数，而是通过识别显卡的世代(Architecture)、层级(Tier)和显存带宽属性进行推算。
   公式： GpuGamingScore = (BaseTier \* GenMultiplier) + VramBonus - ResolutionPenalty
   层级定位 (BaseTier)：通过型号中的数字后缀识别（如 3080，4060）。
   90/Ti: 100分 | 80: 85分 | 70: 70分 | 60: 55分 | 50: 40分 | 50以下/核显: 15-30分。
   世代系数 (GenMultiplier)：代表架构的 IPC 提升和新技术（如 DLSS 3, 光追）。
   RTX 40 / RX 7000: 1.3
   RTX 30 / RX 6000: 1.1
   RTX 20 / RX 5000: 0.9
   GTX 10 / RX 500: 0.7
   显存加成 (VramBonus)：
   VRAM >= 12GB: +10分 (支撑 4K 材质)
   VRAM < 6GB: -15分 (爆显存会导致严重卡顿)
   移动端修正：型号含 "Laptop" 或 "Mobile" 时，分数 × 0.88。
2. CPU 游戏同步率 (消除瓶颈)
   游戏更看重单核频率和缓存大小，而不是核心数。
   公式： CpuGamingScore = (SingleCoreBase \* GenWeight) + CacheBonus
   单核基础分：MaxClockFreqMhz / 60 (例如 5000MHz -> 83分)。
   代际权重 (GenWeight)：
   最新 2 代 (Intel 13/14, Ryzen 7000): 1.1
   主流 (Intel 10-12, Ryzen 3000/5000): 1.0
   老旧 (Intel 9代以前): 0.8
   缓存补偿 (L3 Cache)：
   如果型号含 "X3D" (AMD 3D 缓存) 或主频极高，加 5-10 分。
3. 内存与存储 (游戏平滑度)
   容量阈值：
   < 8GB: 极差 (30分)，会导致频繁掉帧。
   16GB: 优秀 (85分)，目前的甜点位。
   32GB+: 卓越 (100分)。
   加载速度 (Disk)：
   NVMe: 100分 | SATA SSD: 80分 | HDD: 40分 (导致转场加载极慢)。
4. 最终游戏评分公式 (GPM)
   游戏分不是简单的平均分，而是要体现显卡主导制和木桶效应：
   FinalGameScore=(GPU×0.65+CPU×0.20+RAM×0.10+Disk×0.05)×BottleneckFactor
   瓶颈因子 (BottleneckFactor)：如果 GPU 远强于 CPU，或者内存极小，总分会额外下调 10%，因为高端显卡无法全功率发挥。
